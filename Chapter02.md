# 02 변수와 자료형

## 변수와 값

```professer="Sungchul Choi"```

-> "같다"가 아닌 "professer"라는 변수에 "Sungchul Choi"라는 값을 넣으라는 것(할당)

## 변수와 메모리 
#### 프로그래밍에서의 변수 = 어떠한 값을 저장하는 장소
* 메모리 : 변수를 저장하는 물리적 장소
* 메모리 주소 : 이 변수의 저장 위치
* <b>메모리 = RAM</b> -> 휘발성이 있으며 메모리 공간은 반드시 주소값을 가진다.
* 변수 : 프로그램에서 특정한 값을 저장하는 공간의 이름을 뜻한다.

## 변수명
#### 다른 사람이 읽어도 이해할 수 잇는 이름으로 변수명을 선언해야함
일반적으로 알파벳, 숫자, _로 선언

## 메모리 공간
#### 컴퓨터는 0과 1의 이진수를 사용
이진수 한자리를 <b>비트</b>->8개의 비트는 <b>1바이트</b>->1024바이트는 <b>1킬로 바이트</b>-> 1024킬로바이트는 <b>1메가바이트<b>이다.

## 기본 자료형 
1. 정수형 : ```int```
2. 실수형 : ```float```
3. 문자형 : ```str```
4. 불린형 : ```bool```

-> 파이썬은 동적 파이팅으로 변수의 자료형을 인터프리터가 판단할 수 있다.

## 간단한 연산
* 연산자 : + - * / 기호
* 피연산자 : 연산자에 의해 계산되는 숫자

1. 제곱승 ```**```
2. 나눗셈의 몫과 나머지 ```//```,```%```
3. 증가 연산과 감소 연산 ```a+=1```.```a-=1```

## 자료형 변환
* 실수 + 실수 -> 내림이 자동으로 실행
* 동적 타이핑 때문에 형 변환이 자동으로 일어난다.
* 자료형 확인하는 방법 ```type()```
